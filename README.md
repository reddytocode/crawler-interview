# Aestheticart Supplier Crawler
Technical Challenge - Aestheticart Supplier Crawler


## Requirements

- Python 3.11+
- uv package manager

## Installation

- Run scrapper
```bash
uv run main.py
```

- Run scrapper on mocked "All products" page, but fetching each product (73 products)
```bash
uv run mocked_main.py
```

- Run tests:
```bash
uv run pytest .
```

- Run linters:
```bash
uv run ruff format .
```
